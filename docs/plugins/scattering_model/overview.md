# Scattering model plugin

The strength of an acoustic wave scattered from an interface depends on many factors,
including the strength of the incident wave, the properties of the interface and the
frequency of the wave. There are numerous models of this scattering strength. A
scattering model plugin implements one of these models so it can be applied to the waves
scattered from each simulated target.
