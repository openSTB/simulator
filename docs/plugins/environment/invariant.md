# Invariant environment

**Plugin name**: `invariant`
<br>
**Implementation**: [`openstb.simulator.environment.invariant.InvariantEnvironment`][]

This plugin models an environment with time- and space-invariant properties (i.e., a
homogeneous environment). The value of each environmental property must be provided to
the plugin when it is created.
