# Version 0.8.1 (24 June 2026)

## New features

* Added the [openstb.simulator.target.points.target_chunk_iterator][] helper function to
  iterate over chunks of point targets from multiple plugins. A similar function was
  previously part of the private code of the simple point simulation controller.

* Added the [openstb.simulator.util.quaternion_from_vectors][] function to calculate the
  quaternion needed to rotate one vector to another. A similar approach was previously
  used internally to the linear trajectory plugin.


## Improvements

* Add documentation on a number of plugin types.

* Give a clearer error message if a local (unregistered) plugin could not be loaded.
  This is either because the specified module is not installed or specified file does
  not exist. Previously the error message was not obvious about the cause of failure.


## Fixes

* Updated some tests to work around regressions in external libraries.

* Fixed some type hinting.

* Hide test files from the test coverage report.
